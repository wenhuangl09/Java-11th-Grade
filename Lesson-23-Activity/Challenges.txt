Challenge 2:
Write a function total that accepts an array of prices and returns the sum of prices plus tax. Use a tax rate of 8.525%.

To test the function, create an array of prices in the init function.

Challenge 3:
Create a function isin that accepts an array of integers and a integer number and return true if the number is in the array otherwise return false

Create an array of integers to test the function isin.


Challenge 4:

HINT: make use of the function isin from challange 3.

To test the function, create an array representing your lotto number picks and another array representing the lotto numbers drawn
